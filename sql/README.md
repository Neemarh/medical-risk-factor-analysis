# Osteoporosis SQL Analysis Project

## What is this?
I analyzed patient data to find out what factors increase the risk of osteoporosis.

## Files in this project
- `osteoporosis_analysis.sql` - All my SQL queries
- `osteoporosis.csv` - The patient data

## What I learned
- Age is the biggest risk factor (100% of patients over 50 had osteoporosis)
- Corticosteroid medications increase risk in younger patients
- Being underweight + low calcium = higher risk

## How to use
1. Download the files
2. Import the CSV into any database
3. Run the SQL queries
